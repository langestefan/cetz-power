import { Tabs, TabItem } from '@astrojs/starlight/components';

<Tabs>
  <TabItem label="Typst Universe">
    Once published, import the package by its versioned spec:

    ```typst
    #import "@preview/cetz-power:0.1.0" as pg
    ```
  </TabItem>
  <TabItem label="Local checkout">
    For local development, clone the repository and import directly from the
    path:

    ```typst
    #import "path/to/cetz-power/src/lib.typ" as pg
    ```
  </TabItem>
</Tabs>

The library exports the symbol families `bus` (and `bus-frac`),
`wire` / `elbow`, `external-grid`, `transformer`, `load`, `pv-panel`,
and `machine`, plus the composition helper `multi-wire`. The canvas
wrapper `pg.diagram(...)` installs the default style dictionary and
handles HTML output — **always start a diagram with it** rather than
calling `cetz.canvas` directly. See [Your first diagram](/getting-started/first-diagram/)
for what it does and why it matters.
